# **App Name**: Danny Store

## Core Features:

- Product Browsing & Filtering: Browse products with advanced filtering (category, price, size, color, brand) and sorting options.
- Virtual Try-On: Enable users to virtually try on clothes using uploaded photos and avatar models.  Simulate try-on experience by displaying placeholders for different states (loading, success, error).
- WhatsApp Checkout: Automatically generate a formatted WhatsApp message containing order details (product list, images, subtotal, delivery, total) and customer information.
- Customer Account Management: Allow users to manage their profile (name, email, WhatsApp number, language), view order history, and manage product reviews.
- Product Reviews and Ratings: Enable users to submit product reviews with ratings, and view overall product ratings and community reviews.
- AI-Powered Outfit Suggestions: Suggest additional complementary items from the catalog to the user, using a generative AI tool. This feature will recommend items to go with the item they are viewing or which are already in the cart, based on style and fit.

## Style Guidelines:

- Light Theme: Background color: Light gray (#F5F5F5), Primary color: Deep Blue (#1A237E), Accent color: Gold (#FFD700).
- Dark Theme: Background color: Dark Gray (#121212), Primary color: Light Blue (#BBDEFB), Accent color: Gold (#FFD700).
- Headline font: 'Playfair', a serif font for elegance and fashion. Body font: 'PT Sans', a humanist sans-serif, to ensure legibility.
- Use premium, minimalist icons relevant to product categories and user actions. Ensure icons are consistent and scalable.
- Employ a modern, mobile-first responsive design with a clear visual hierarchy and consistent spacing.
- Implement subtle animations (hover effects, transitions) to enhance user experience and provide visual feedback.
